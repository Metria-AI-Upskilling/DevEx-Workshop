# Deep Dives

This folder contains advanced topics that go beyond the fundamental AI development workflows covered in the main course modules.

## What's Inside

These stand-alone modules explore more specialized techniques and use cases for AI-powered development:

- **DD1. [Advanced Planning with Speckit](DD1-advanced-planning-speckit.md)** - Go beyond simple planning with systematic spec-driven development. Learn how SpecKit analyzes codebases, enforces consistency, and breaks complex features into executable tasks.

- **DD2. [QA and Pull Requests](DD2-qa-and-pull-requests.md)** - Scale your development with automated code review and PR workflows. Learn how AI-assisted review catches different errors than manual review and handles the full PR lifecycle.

- **DD3. [Transforming & Structuring Data](DD3-transforming-structuring-data.md)** - Learn how AI transforms unstructured qualitative data (interviews, feedback, support tickets) into structured, actionable insights. Perfect for product managers, designers, and business analysts who need to extract patterns from large volumes of text.

- **DD4. [Language Server Protocol](DD4-language-server-protocol.md)** - Understand how LSP gives agents semantic code understanding instead of text-based pattern matching. Learn why this enables safe refactoring and accurate code analysis.

- **DD5. [Task Orchestration with VibeKanban](DD5-task-orchestration-vibekanban.md)** - Coordinate multiple agents working in parallel with ticket-driven development. Learn how to break work into discrete tasks, track progress, and manage dependencies.

## When to Use These

The Deep Dives modules are optional and can be explored:
- After completing the main foundation modules
- When you encounter a specific use case that matches one of these topics
- As reference material when you need to apply these techniques in your work

These modules assume you're comfortable with the core concepts from modules 1-5 in the main Course Material folder.
