# Extra Features

This folder contains continuation exercises that build upon the map application developed in the main course modules. These are not standalone lessons but rather practical feature implementations that extend the app further.

## What's Inside

These modules guide you through adding real features to your map application:

- **EF1. [Search & Live Filter](EF1-search-and-filter.md)** - Add live text search that filters markers by name
- **EF2. [Marker Colours](EF2-marker-colours.md)** - Implement a per-marker color picker with visual presets
- **EF3. [Import & Export](EF3-import-export.md)** - Add GeoJSON import/export functionality
- **EF4. [Distance Tool](EF4-distance-tool.md)** - Build an interactive tool to measure distances between markers

## How to Use These

These modules are designed to be completed **after** the main foundation modules (1-5). Each one:
- Builds directly on the map app you've been developing
- Adds a complete, working feature
- Demonstrates practical patterns for working with AI on real features
- Can be completed in any order (they're independent of each other)

Think of these as your "capstone" exercises - putting the AI development skills you've learned into practice on realistic feature work.
