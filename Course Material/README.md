# Metria DevEx Workshop

Welcome to the Metria DevEx Workshop! This collection of AI development modules is intended for engineers to upskill within AI Assisted Development. The modules cover the basics in such a way that they should be accessible for other roles as well (e.g. design, product, devops).

## Course Structure

This bootcamp is organized into modules that build on each other. Each module teaches a specific AI workflow concept while adding a real feature to the demo map app.

### Foundation Modules

These core modules teach the fundamental concepts of AI-empowered development:

1. **[Getting Started](01-getting-started.md)**
   - Prerequisites, setup, installing Claude Code
   - Exercise: Run the Map app locally

2. **[Working with AI Agents](02-working-with-ai-agents.md)**
   - Agents vs chat, the 80/20 planning principle, context management
   - Exercise: Add localStorage persistence so markers survive page refresh

3. **[Skills & Slash Commands](03-skills-and-slash-commands.md)**
   - What skills are, built-in commands, writing custom reusable skills
   - Exercise: Write a custom skill for a workflow you actually use

4. **[Model Context Protocol (MCP)](04-model-context-protocol.md)**
   - What MCP is, how it connects agents to external tools and data
   - Exercise: Add marker clustering using Context7 MCP to fetch live docs

5. **[Safety and Guardrails](05-safety-and-guardrails.md)**
   - Why autonomous agents need guardrails, SafetyNet, custom rules
   - Exercise: Test SafetyNet blocking a destructive git operation

### Additional Resources

- **[Deep Dives](Deep%20Dives/)** - Advanced topics exploring specialized techniques like data transformation and structuring
- **[Extra Features](Extra%20Features/)** - Continuation exercises that add more features to your map app (search, colors, import/export, distance tool)

## Next Steps

Start with [Module 01: Getting Started](01-getting-started.md) and work through the foundation modules (1-5) in order. After completing the foundation, you can explore the [Deep Dives](Deep%20Dives/) and [Extra Features](Extra%20Features/) in any order based on your interests.
